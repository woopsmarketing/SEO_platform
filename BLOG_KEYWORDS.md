# SEO월드 블로그 키워드 목록

> 니치 공식: 시장 x 대상 x 문제 x 행동 x 상황
> Level 4(돈 되는 구간)부터 Level 3 순으로 정렬
> 하루 2~3편 발행, 위에서부터 순서대로 쓰면 됨

## 발행 가이드

- 1번부터 순서대로 `/write-blog [키워드]`로 발행
- 하루 2~3편, 위에서 아래로
- #1~30: 첫 2주 집중 (Level 4, 빠른 색인 + 도구 전환)
- #31~45: 3주차 (Level 3 낮음, 트래픽 확보)
- #46~78: 4주차~ (Level 3 중간, 도메인 신뢰도 필요)
- 78개 소진 후 니치 공식으로 추가 생성
