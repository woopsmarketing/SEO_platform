# SEO월드 블로그 키워드 목록

> 니치 공식: 시장 x 대상 x 문제 x 행동 x 상황 (Level 3~4 키워드만)
> 하루 2~3편 발행 기준, 약 1개월분 분량

## 발행 우선순위 (도구 직결 + 난이도 낮음)

1. robots.txt 작성 방법 예시 포함
2. sitemap.xml 만드는 법
3. 메타태그 설정하는 방법
4. 백링크 확인하는 방법 무료
5. 키워드 밀도 적정 수준 몇 퍼센트
6. 내 사이트 구글 검색 안되는 이유
7. dofollow 백링크와 nofollow 차이
8. 티스토리 블로그 SEO 설정 방법
9. 블로그 글 키워드 어디에 넣어야 하나
10. 카페24 쇼핑몰 SEO 설정 방법
