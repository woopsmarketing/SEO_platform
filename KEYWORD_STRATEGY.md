## 4. VebAPI 연동 구현 계획

### 4.1 백링크 분석기 (`/tools/backlink-checker`)

```
사용자가 도메인 입력
→ POST /api/backlink-check
→ VebAPI GET /seo/backlinkdata?website={domain}
→ 결과 파싱:
   - 총 백링크 수, doFollow 수, 참조 도메인 수
   - 백링크 목록 (소스 URL, 앵커 텍스트, nofollow 여부, 권위 점수)
→ analyses 테이블에 저장 (tool_type: "backlink-checker")
→ 프론트엔드 표시:
   - 통계 카드 (총 백링크, doFollow, 참조 도메인)
   - 백링크 목록 테이블 (정렬, 필터)
```

**환경변수**: `VEBAPI_KEY`

### 4.2 구글 순위 확인 (`/tools/rank-checker`)

```
사용자가 키워드 + 지역(한국) 입력
→ POST /api/rank-check
→ VebAPI GET /serp/googleaimode?q={keyword}&geo=South Korea&device_type=desktop_chrome
→ 결과 파싱:
   - AI 요약 응답 (response_text)
   - 인용 소스 (citations + urls)
   - 관련 링크 (links)
→ analyses 테이블에 저장 (tool_type: "rank-checker")
→ 프론트엔드 표시:
   - AI 요약 결과
   - 인용된 사이트 목록 (내 사이트가 포함되어 있는지 하이라이트)
   - 관련 링크 목록
```

### 4.3 Rate Limit

VebAPI 무료 플랜: 50 requests/월 → 매우 제한적
→ Free 유저: 1회/일, Pro 유저: 10회/일 권장
→ 또는 Scout($9/월, 10,000건) 플랜 가입 후 제한 완화
